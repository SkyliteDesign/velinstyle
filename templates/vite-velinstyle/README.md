# VelinStyle Vite starter

1. From the VelinStyle repository root, build the framework: `npm install && npm run build`.
2. Copy this folder outside the monorepo or run `npm install` here (dependency `velinstyle` points to `file:../..`).
3. `npm run dev` — open the printed localhost URL.

Includes three static HTML pages, a skip link, navigation, and `<velin-theme-toggle>`.
