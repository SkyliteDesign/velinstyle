The CSS Framework That
Makes Bootstrap Sweat

Accessibility-first, zero-JS components, OKLCH colors, Container Queries, View Transitions — everything modern CSS can do, in one framework.

Get Started View on GitHub

~95 KB minified CSS, 31 KB JS — lighter than Bootstrap's CSS alone

Built for the Modern Web

Every feature designed around what CSS can do today — not five years ago.

Accessibility First

WCAG AA contrast, focus management, screen-reader support, and forced-colors mode baked into every component.

OKLCH Color System

Perceptually uniform colors, automatic dark-mode palette, and color-mix() utilities out of the box.

Container Queries

Components adapt to their container, not the viewport. True component-level responsive design.

Zero-JS Components

24+ CSS-only components that work without a single line of JavaScript. Progressive enhancement at its finest.

Web Components

14 interactive elements with Shadow DOM encapsulation, focus trapping, and keyboard navigation built in.

Theme Presets

Sharp, Soft, Brutalist — or create your own with CSS custom properties. Dark mode is automatic.

Up and Running in Seconds

Drop two lines into your HTML and start building.

CDN Install 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@birdapi/velinstyle@0.4.0/dist/velinstyle.css">
<script type="module" src="https://cdn.jsdelivr.net/npm/@birdapi/velinstyle@0.4.0/dist/velinstyle-components.js"></script>
Starter Template 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@birdapi/velinstyle@0.4.0/dist/velinstyle.css">
  <script type="module" src="https://cdn.jsdelivr.net/npm/@birdapi/velinstyle@0.4.0/dist/velinstyle-components.js"></script>
</head>
<body>
  <div class="velin-container">
    <article class="velin-card">
      <div class="velin-card__body">
        <h2 class="velin-card__title">Hello VelinStyle</h2>
        <p class="velin-card__text">You're ready to build accessible, beautiful UIs.</p>
        <button class="velin-btn velin-btn--primary">Let's Go</button>
      </div>
    </article>
  </div>
</body>
</html>
