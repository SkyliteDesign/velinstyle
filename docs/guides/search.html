VelinSearch

Central framework search with fuzzy matching, keyboard navigation, and offline JSON index.

Web Component

<velin-search index="/search-index.json" categories="docs,components,api,examples"></velin-search>

JavaScript API

import { velinSearch, registerSearchProvider } from '@birdapi/velinstyle/search';

await velinSearch.loadIndex('/search-index.json');
registerSearchProvider('app', () => [{ id: '1', title: 'Custom', url: '/x', category: 'examples' }]);
const { results, groups } = await velinSearch.query('modal', { fuzzy: 0.2 });

Build index

npx velinstyle search index --out dist/search-index.json
npm run docs:generate   # also writes dist/search-index.json
