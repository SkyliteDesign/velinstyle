Responsive Layout Audit 0.8.0

Statische Prüfung von HTML auf typische Flex-, Grid- und Breakpoint-Fehler — mit Vorschlägen für Smartphone, Tablet und Desktop.

Befehle

npx velinstyle layout audit ./samples
npx velinstyle layout suggest page.html
npx velinstyle layout fix page.html --dry-run
npx velinstyle layout fix page.html --write

--json liefert maschinenlesbare Reports für CI oder Agenten.

Sichere Auto-Fixes

layout fix --write wendet nur eine Whitelist an: z. B. velin-flex--wrap ergänzen, velin-hidden mit velin-md-block paaren. Kein strukturelles DOM-Rewriting.
