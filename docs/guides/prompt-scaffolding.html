Prompt Scaffolding 0.8.0

Erzeuge Navbars, Cards, Modals und Seitenlayouts aus einer Textbeschreibung — deterministisch über Blueprint-Komposition, ohne API-Schlüssel.

Befehle

npx velinstyle scaffold list-intents
npx velinstyle scaffold "Navbar mit Logo und Suchfeld" -o nav.html
npx velinstyle scaffold "Dashboard mit Sidebar" --json

Unterstützte Intents

IntentBeispiel-PromptBlueprints
navbarNavigation mit Suchenavbar-header (+ search-field)
modalBestätigungs-Dialogmodal
card3 Karten im Gridcard-grid
dashboardAdmin Shellnavbar-header, layout-dashboard
loginLogin Formularform-login
pricingPreistabellepricing-table

Rezepte: cli/scaffold-recipes.json. Unbekannte Prompts fallen auf hero-section zurück (confidence: low).

Workflow

  1. velinstyle scaffold "…" -o snippet.html
  2. velinstyle layout suggest snippet.html — Responsive-Hinweise
  3. velinstyle scan snippet.html — A11y & Security
  4. Manuell anpassen, dann in dein Projekt übernehmen

Grenzen

  • Nur velin-*-Klassen und bestehende Blueprints — kein freies CSS.
  • Kein LLM im CLI; Cursor/Agenten können --json nutzen und Ergebnisse verfeinern.
  • Slots: brand, title, cta werden heuristisch aus dem Prompt extrahiert.
