Motion & HTML attributes

Declarative velin-* attributes bridge to Web Components and the motion runtime.

import { bootFromDOM } from '@birdapi/velinstyle/runtime';
import { initMotion } from '@birdapi/velinstyle/motion';

bootFromDOM(document, { attributes: true });
initMotion();

  • velin-modal / velin-tabs — lazy-load Web Components
  • velin-code / velin-quote — content helpers
