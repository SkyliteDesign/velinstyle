VelinHighlight

Token-based syntax highlighting without Prism or other CDN dependencies. Uses OKLCH token colors from highlight.css and lazy-loads lexers when blocks scroll into view.

JavaScript API

import { velinSyntax, initHighlight } from '@birdapi/velinstyle/highlight';

initHighlight(document);
// or: velinSyntax.highlightElement(document.querySelector('pre'));

HTML attribute

<pre velin-code="js">const x = 42;</pre>

Web component

.velin-btn { padding: 0.5rem 1rem; } .velin-btn--primary { background: var(--velin-color-primary); }

Languages: JavaScript, TypeScript, HTML, CSS, JSON, Markdown, Shell, SQL.
