Velin-Meta 1.0.0

Agent-optimized metadata for Cursor, Copilot, and custom tools — not hidden from humans, but structured for machines.

Overview

Velin-Meta is not encryption or KI-only access. Any client can fetch velin-agent.json. It gives assistants a compact snapshot of VelinStyle: components, CLI, conventions, and links to generated reference.

Files and MIME

  • dist/velin-agent.json — canonical bundle
  • dist/llms.txt — short index (llms.txt convention)
  • MIME: application/vnd.velinstyle.meta+json
npx velinstyle meta
npx velinstyle meta --base-url https://velinstyle.info
npx velinstyle meta page my-page.html --write

Page-level meta

<script type="application/vnd.velinstyle.meta+json" id="velin-meta">
{ "page": { "intent": "component-doc" }, "allowed": { "classesPrefix": ["velin-"] } }
</script>

Cursor / agents

Point your agent at dist/velin-agent.json or add to project rules:

When editing VelinStyle HTML, follow conventions in dist/velin-agent.json.
Use velin-* classes only. Run velinstyle scan on new markup.
