Vite & React

Zwei Wege: statisches Vite + HTML (templates/vite-velinstyle) oder React 18 mit @velinstyle/react (templates/vite-react-velinstyle).

React-Starter (0.7.5)

cd templates/vite-react-velinstyle
npm install
npm run dev

Im Monorepo verweist package.json auf file:../.. (VelinStyle) und file:../../packages/react.

@velinstyle/react bauen

cd packages/react
npm install
npm run build

App-Einstieg

import { VelinThemeToggle, VelinSheet } from '@velinstyle/react';
import 'velinstyle/dist/velinstyle.min.css';
import 'velinstyle/dist/velinstyle-components.min.js';

Wrapper sind dünne React.forwardRef-Hüllen um native Custom Elements — keine duplizierte Logik.
